                        3. MANAGING STUDENTS
================================================================================

A. HOW TO ADD A NEW STUDENT
---------------------------

1. Click "Students" in left sidebar
2. You'll see "Add New Student" form at top
3. Fill in the form:

   ACCOUNT INFORMATION:
   - Full Name: Rahul Kumar
   - Username: rahul_kumar (for student login)
   - Email: rahul@email.com
   - Password: student123 (student will use this to login)
   
   ACADEMIC INFORMATION:
   - Class: Select the class (e.g., Class 5 - A)

   - Blood Group: A+, B+, O+, etc.
   
   PARENT INFORMATION:
   - Father's Name: Mr. Kumar

   - Phone: Student's phone (optional)
   - Address: Full home address
   - Previous School: Name of previous school

4. Click "Add Student"

5. IMPORTANT: The system will automatically generate:
   - Roll Number (e.g., 2024050001)
   - Admission Number (e.g., ADM20240001)
   
   Write these down or print the admission form!

B. HOW TO VIEW ALL STUDENTS
---------------------------

1. Click "Students" in left sidebar
2. Scroll down to see "All Students" table
3. You can see:
   - Roll Number
   - Admission Number
   - Name
   - Class
   - Gender
   - Guardian Phone
   - Status (Active/Inactive)

C. HOW TO SEARCH FOR A STUDENT
------------------------------

1. Go to Students page
2. In the search box, type:
   - Student name, OR
   - Roll number, OR
   - Admission number
3. Click the search button
4. Matching students will appear

D. HOW TO FILTER STUDENTS BY CLASS
----------------------------------

1. Go to Students page
2. Use the "Class" dropdown
3. Select a class (e.g., Class 5)
4. Click search button
5. Only students from that class will show

E. HOW TO EDIT A STUDENT
------------------------

1. Find the student in the list
2. Click the blue "Edit" button (pencil icon)
3. The form will fill with student's current data
4. Make your changes
5. Click "Update Student"

F. HOW TO DELETE A STUDENT
--------------------------

WARNING: This permanently deletes the student and all their data!

1. Find the student in the list
2. Click the red "Delete" button (trash icon)
3. Confirm when asked "Are you sure?"
4. Student will be deleted

G. HOW TO PRINT ADMISSION FORM
------------------------------

1. Find the student in the list
2. Click the green "Print" button (printer icon)
3. Admission form opens in new window
4. Click "Print Admission Form" button at top
5. Print dialog appears
6. Select your printer
7. Click Print

The admission form includes:
- School header with logo
- Student photo placeholder
- All student information
- Parent information
- Previous school information
- Declaration section
- Signature lines

                        7. FEE MANAGEMENT
================================================================================

A. UNDERSTANDING FEE STRUCTURE
------------------------------

The system supports:
- Monthly fees (Tuition)
- One-time fees (Admission)
- Quarterly fees
- Annual fees
- Transport fees
- Other fees

B. HOW TO GENERATE MONTHLY FEES FOR A CLASS
-------------------------------------------

This creates fee records for ALL students in a class at once.

1. Click "Fee Management" in sidebar
2. Click "Generate Fees" tab
3. Fill in:
   - Class: Select class (e.g., Class 5 - A)
   - Month: January 2025
   - Academic Year: 2024-2025
   - Due Date: 10-01-2025 (when fee should be paid)
4. Click "Generate Fees for All Students"
5. System creates fee record for each student

C. HOW TO CREATE INDIVIDUAL FEE
-------------------------------

For one student only:

1. Click "Fee Management" in sidebar
2. Click "Create Individual Fee" tab
3. Fill in:
   - Student: Search and select student
   - Fee Type: Select type (or custom)
   - Amount: Enter amount
   - Month: January 2025
   - Academic Year: 2024-2025
   - Due Date: Select date
4. Click "Create Fee Record"

D. HOW TO VIEW FEE STATUS
-------------------------

1. Go to Fee Management
2. You'll see a table with all fees:
   - Student name
   - Class
   - Month
   - Amount
   - Due Date
   - Status (Paid/Unpaid/Partial)

E. HOW TO FILTER FEES
---------------------

Use the filters at top:
- Status: All / Paid / Unpaid / Partial
- Class: Filter by class
- Click "Filter" to apply

F. UNDERSTANDING FEE STATUS
---------------------------

- UNPAID (Red): No payment received
- PARTIAL (Yellow): Some amount paid, balance remaining
- PAID (Green): Full amount received

                        8. COLLECTING PAYMENTS
================================================================================

A. HOW TO COLLECT FEE PAYMENT
-----------------------------

METHOD 1 - From Fee Management Page:

1. Go to "Fee Management"
2. Find the student's fee record
3. Click green "Collect" button
4. Enter payment details:
   - Amount to Pay: Enter amount
   - Payment Date: Today's date
   - Payment Method: Cash/Card/Bank Transfer/UPI
   - Transaction ID: (for digital payments)
   - Remarks: Any notes
5. Click "Collect Payment"
6. Receipt number is generated automatically

METHOD 2 - From Collect Payment Tab:

1. Go to "Fee Management"
2. Click "Collect Payment" tab
3. Select student with pending fee from dropdown
4. Fill in payment details
5. Click "Collect Payment"

B. HOW TO PRINT FEE RECEIPT
---------------------------

After collecting payment:

1. Go to "Payments" in sidebar

                        11. RESULT MANAGEMENT
================================================================================

A. HOW TO ENTER MARKS
---------------------

1. Click "Results" in sidebar
2. Select:
   - Exam: Choose exam
   - Class: Choose class
   - Subject: Choose subject
3. Click "Load Students"
4. Student list appears with mark entry fields
5. Enter marks for each student:
   - Theory Marks
   - Practical Marks (if applicable)
   - Check "Absent" if student was absent
6. Click "Save Marks"

B. AUTOMATIC GRADE CALCULATION
------------------------------

The system automatically calculates:
- Total marks
- Percentage
- Grade (A+, A, B+, B, C, D, F)
- Pass/Fail status

Grade scale:
- 90-100%: A+ (Outstanding)
- 80-89%: A (Excellent)
- 70-79%: B+ (Very Good)
- 60-69%: B (Good)
- 50-59%: C+ (Above Average)
- 40-49%: C (Average)
- 33-39%: D (Below Average)
- Below 33%: F (Fail)

C. HOW TO PRINT RESULT CARD
---------------------------

1. After entering all subject marks
2. Find student in results
3. Click "Print Report Card"
4. Report card shows:
   - Student details
   - All subject marks
   - Total and percentage
   - Grade
   - Remarks
   - Signature lines

D. HOW TO VIEW CLASS RESULTS
----------------------------

1. Go to Results page
